📩 Spam Detector – AI Powered Web App

An AI-powered Spam Detection Web Application built using Django, Natural Language Processing (NLP), and Multinomial Naive Bayes from scikit-learn.

This project classifies text messages as:

⚠️ Spam

✅ Ham (Not Spam)

🚀 Features

🧠 Machine Learning-based spam classification

📊 NLP text preprocessing

🎨 Modern animated UI

🔒 CSRF protected Django form

⚡ Fast prediction using Multinomial Naive Bayes

🛠️ Tech Stack

Backend: Django

Machine Learning: scikit-learn

Algorithm: Multinomial Naive Bayes

Frontend: HTML, CSS

Language: Python

⚙️ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/priyankabhendale/spam-detector.git
cd spam-detector

2️⃣ Create virtual environment (Recommended)
python -m venv venv
venv\Scripts\activate   # Windows

3️⃣ Install dependencies
pip install -r requirements.txt

If you don’t have requirements.txt, install manually:

pip install django scikit-learn pandas numpy
4️⃣ Run the server
python manage.py runserver

Open in browser:

http://127.0.0.1:8000/
